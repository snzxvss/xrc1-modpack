# Assets Folder

Coloca aquí los recursos gráficos del instalador:

## Archivos necesarios:

### 1. xrc1-logo.png
- **Tamaño recomendado:** 400x400px o 512x512px
- **Formato:** PNG con transparencia
- **Uso:** Logo principal en la interfaz

### 2. xrc1-logo-small.png
- **Tamaño recomendado:** 40x40px o 64x64px
- **Formato:** PNG con transparencia
- **Uso:** Logo en la barra de título

### 3. icon.ico (Windows)
- **Tamaño:** 256x256px
- **Formato:** ICO (ícono de Windows)
- **Uso:** Ícono de la aplicación en Windows

### 4. icon.png (Linux)
- **Tamaño:** 512x512px
- **Formato:** PNG
- **Uso:** Ícono de la aplicación en Linux

### 5. icon.icns (macOS)
- **Tamaño:** Múltiples tamaños
- **Formato:** ICNS (ícono de macOS)
- **Uso:** Ícono de la aplicación en macOS

## Cómo crear los íconos

### Para Windows (.ico):
Usa herramientas online como:
- https://icoconvert.com/
- https://convertio.co/png-ico/

### Para macOS (.icns):
Usa `png2icns` o herramientas online:
- https://cloudconvert.com/png-to-icns

### Para extraer de la imagen unnamed.jpg:
1. Abre la imagen en un editor (Photoshop, GIMP, etc.)
2. Recorta el logo que desees
3. Redimensiona a los tamaños necesarios
4. Exporta en los formatos correspondientes

## Temporalmente

Si no tienes los logos listos, el instalador usará placeholders. La funcionalidad seguirá funcionando sin los archivos de imagen.
